# sobel -> calculate the gradient of the image in both x and y directions using the Sobel operator
cv.Sobel(src, ddepth, dx, dy, ksize=3)

ksize	ksize	description
1	    1×3	    simple gradient, no smoothing,not use gaussian blur, very sensitive to noise
3	    3×3	    default, good balance between noise reduction and edge detection, commonly used
5	    5×5	    more smoothing, less sensitive to noise, but also less accurate edge detection, may miss finer details
7	    7×7	    more smoothing, less sensitive to noise, but also less accurate edge detection, may miss finer details

kernel function defaults given by OpenCV:
1 * 3
X: [-1  0  1]  (one dimension，no smoothing, very sensitive to noise)
Y: [-1  0  1]^T
3*3
X :    [-1  0  1]
       [-2  0  2]
       [-1  0  1]
Y:
-1 -2 -1
 0  0  0
 1  2  1


轮廓往左/右移动”取决于你用的外力定义：
若外力是 +∇I：会朝更亮方向走（dx<0 往左，dx>0 往右）。
若外力是 -∇I：会朝更暗方向走（相反）
外力 = 能量场的梯度
外力 F = ∇E
轮廓移动方向：向能量降低的方向 = -∇E
外力定义	表达式	移动方向	含义
+∇I	正梯度	朝向更亮	向亮度增加方向
-∇I	负梯度	朝向更暗	向亮度减小方向
1. 情况1：+∇I（正梯度力）

 dx > 0	正梯度	朝右	轮廓向右移动
    图像：暗 ────→ 亮
          左       右
    
    梯度方向：→（指向更亮）
    外力 = +∇I：→
    轮廓移动：→（朝向更亮）
    
    dx=+100 → 轮廓向右移动
dx < 0 → 轮廓向左移动
    图像：亮 ←──── 暗
          左       右
    
    梯度方向：←（指向更亮）
    外力 = +∇I：→
    轮廓移动：→（朝向更亮）
    
    dx=-100 → 轮廓向左移动
2. 情况2：-∇I（负梯度力）
dx > 0（向右变亮）：
    图像：暗 ────→ 亮
      左       右

    梯度方向：→
    外力 = -∇I：←（相反）
    轮廓移动：←（朝向更暗）
    
    dx=+100 → 轮廓向左移动
dx < 0（向右变暗）：
    图像：亮 ←──── 暗
          左       右
    
    梯度方向：←
    外力 = -∇I：→（相反）
    轮廓移动：→（朝向更暗）
    
    dx=-100 → 轮廓向右移动

E = dx² + dy²
E_ext = -E
E_ext 场的分布示例：

位置：   -10   -5   0   -5   -10
         ←──────────────────→
           A    B   C    D   E

计算各点的 fx：
在 B 点：
左边值：-10（A点）
右边值：0（C点）

fx = 右边 - 左边 = 0 - (-10) = +10  ← 正值
在 D 点：
左边值：0（C点）
右边值：-10（E点）

fx = 右边 - 左边 = -10 - 0 = -10  ← 负值







1. left light, right dark
行1：  50   100  150
行2：  50   100  150
行3：  50   100  150

Sobel X核：
-1  0  1
-2  0  2
-1  0  1

计算中心点(100)的dx：
dx = (-1×50) + (0×100) + (1×150) +
     (-2×50) + (0×100) + (2×150) +
     (-1×50) + (0×100) + (1×150)
   
   = -50 + 0 + 150
     -100 + 0 + 300
     -50 + 0 + 150
   
   = 100 + 200 + 100
   = **400**
2. left dark, right light
行1：  150  100  50
行2：  150  100  50
行3：  150  100  50

计算中心点(100)的dx：
dx = (-1×150) + (0×100) + (1×50) +
     (-2×150) + (0×100) + (2×50) +
     (-1×150) + (0×100) + (1×50)
   
   = -150 + 0 + 50
     -300 + 0 + 100
     -150 + 0 + 50
   
   = -100 -200 -100
   = **-400**

6. some edge cases

行2：  255  150  50

计算中心点(150)的dx：
dx = (-1×255) + (0×150) + (1×50) +
     (-2×255) + (0×150) + (2×50) +
     (-1×255) + (0×50) + (1×0)
   
   = -255 + 0 + 50
     -510 + 0 + 100
     -255 + 0 + 0
   
   = -205 - 410 - 255
   = **-870**

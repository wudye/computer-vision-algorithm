部分	含义	物理意义
x, y	当前轮廓点坐标	当前位置
gamma	迭代步长	惯性系数
gamma * x	惯性项	保持当前位置的趋势
fx[i, j], fy[i, j]	外力	向边缘收敛的力
gamma * x + fx	总力向量	惯性 + 外力
inv	逆矩阵	内部能量约束
@	矩阵乘法	应用内部约束
xn, yn	新轮廓点坐标	下一时刻位置
# 理论公式
v_new = (A + γI)⁻¹ · (γ·v_old + F_ext)

# 代码实现
xn = inv @ (gamma * x + fx[i, j])
yn = inv @ (gamma * y + fy[i, j])

# 对应关系：
inv      →  (A + γI)⁻¹
gamma    →  γ
x        →  v_old (X坐标)
y        →  v_old (Y坐标)
fx[i,j]  →  F_ext (X方向外力)
fy[i,j]  →  F_ext (Y方向外力)
xn       →  v_new (X坐标)
yn       →  v_new (Y坐标)
详细计算步骤
# 步骤1：计算惯性项
gamma * x
gamma = 0.01
x = np.array([50, 100, 150])

gamma * x = [0.5, 1.0, 1.5]
物理含义：

保持当前位置的惯性
防止移动过快
时间积分项

# 步骤2：获取外力
fx[i, j]
# 轮廓点位置
i = [75, 101, 125]  # y坐标（已裁剪和转换）
j = [50, 100, 150]  # x坐标（已裁剪和转换）

# 从外部能量场梯度图中取值
fx = np.array([[2.5, 3.1, 2.8, ...],
               [2.0, 3.0, 2.5, ...],
               ...])

fx[i, j] = [fx[75, 50], fx[101, 100], fx[125, 150]]
         = [2.5, 3.0, 2.5]
物理含义：

外部能量场的梯度
引导轮廓向边缘移动的力
每个点的外力向量
# 步骤3：计算总力向量
gamma * x + fx[i, j]

gamma * x = [0.5, 1.0, 1.5]
fx[i, j]  = [2.5, 3.0, 2.5]

total = [0.5, 1.0, 1.5] + [2.5, 3.0, 2.5]
       = [3.0, 4.0, 4.0]
物理含义：
总力 = 惯性力 + 外力
     = 保持当前趋势 + 向边缘收敛

# 步骤4：应用内部约束

inv @ (gamma * x + fx[i, j])
矩阵乘法示例：

假设 n = 3（简化）：
# inv (3×3 矩阵)
inv = [[0.88, 0.50, 0.50],
       [0.50, 0.88, 0.50],
       [0.50, 0.50, 0.88]]

# total (3×1 向量)
total = [3.0, 4.0, 4.0]

# 矩阵乘法
xn = inv @ total
   = [0.88×3.0 + 0.50×4.0 + 0.50×4.0,
      0.50×3.0 + 0.88×4.0 + 0.50×4.0,
      0.50×3.0 + 0.50×4.0 + 0.88×4.0]
   
   = [2.64 + 2.00 + 2.00,
      1.50 + 3.52 + 2.00,
      1.50 + 2.00 + 3.52]
   
   = [6.64, 7.02, 7.02]
物理含义：

应用内部能量约束（弹性+刚性）
相邻点相互影响
平滑轮廓形状


公式的核心含义：
新位置 = 逆矩阵 × (惯性项 + 外力)

其中：
- 逆矩阵：应用内部能量约束（弹性+刚性）
- 惯性项：保持当前位置的趋势
- 外力：引导轮廓向边缘收敛

gamma * y = [0.753, 1.008, 1.251, ...]

gamma * x + fx[i, j]

  惯性项    外力

        = [0.505 + 2.5, 1.007 + 3.0, 1.502 + 2.8, ...]
        = [3.005, 4.007, 4.302, ...]

force_y = gamma * y + fy[i, j]
        = [0.753 + 1.2, 1.008 + 1.5, 1.251 + 1.3, ...]
        = [1.953, 2.508, 2.551, ...]

# 步骤6：应用内部约束
xn = inv @ force_x  # 200×200 矩阵 × 200×1 向量
yn = inv @ force_y

# 结果（新位置）：
xn = [6.64, 7.02, 7.52, ...]
yn = [4.12, 4.35, 4.48, ...]


物理过程：
* 计算趋势：gamma * x（惯性）
* 获取外力：fx[i, j]（边缘吸引）
* 叠加力：gamma*x + fx
* 应用约束：inv @ force（相邻点影响）
* 得到新位置：xn, yn
这是 Snake 算法的隐式迭代公式，通过矩阵求逆同时处理内部约束和外部力，稳定地更新轮廓位置向边缘收敛！

计算总力：gamma*x + fx
为什么还要应用内部约束？
内部约束不是总力的一部分吗？
gamma*x + fx 并不是"总力"！

总力 = 内部力 + 外力
    = F_internal + F_external
gamma * x + fx[i, j]
    ↑         ↑
  惯性项    外力
缺少了内部力！

正确的物理过程
完整的力平衡
方程：γ(v_new - v_old) + A·v_new + F_ext = 0

其中：
- γ(v_new - v_old)：时间惯性项
- A·v_new：内部约束力（弹性+刚性）
- F_ext：外力

A·v_new + γ·v_new = γ·v_old + F_ext

(A + γI)·v_new = γ·v_old + F_ext

v_new = (A + γI)⁻¹ · (γ·v_old + F_ext)
        ↑           ↑          ↑
      内部约束      惯性项    外力
gamma * x
这不是力，而是惯性项（时间积分）

gamma * x 的物理含义：
# 离散化时间演化：
v_new - v_old = dt * (F_total / mass)

简化：
γ(v_new - v_old) = F_total

展开：
γ·v_new - γ·v_old = F_internal + F_external

整理：
(A + γI)·v_new = γ·v_old + F_external

旧位置的惯性影响

A·v_new

xn = (A + γI)⁻¹ · (γ·x + fx)

内部力 + 惯性 = 惯性项 + 外力

整理：
A·xn = γ·(x - xn) + fx
